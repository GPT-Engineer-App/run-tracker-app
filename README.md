# run-tracker-app

Build a web app to help me track my running.

Every run, I want to track what time I ran, how far I ran, and how I felt about the run.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/run-tracker-app.git
cd run-tracker-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
